# Welcome to My Blog!

I'm excited to launch my new blog built with **GitHub Pages** and **Jekyll**. This is the perfect platform for sharing my ideas, projects, and learnings with the world.

## What to Expect

On this blog, you'll find posts about:

- **Development & Coding**: Tips, tricks, and tutorials on software development
- **GitHub**: How to use GitHub effectively for version control and collaboration
- **Web Technologies**: Insights on modern web development practices
- **Projects**: Showcasing interesting projects and learnings

## Why GitHub Pages?

GitHub Pages is an amazing platform because:

1. **Free Hosting**: No need to pay for expensive hosting
2. **Version Control**: All content is version controlled with Git
3. **Simple**: Built on Jekyll, a static site generator
4. **Fast**: Static sites are incredibly fast
5. **Markdown**: Write posts in familiar Markdown format

## Getting Started

If you want to create your own blog using this setup, here's what you need to know:

### Blog Post Format

Every blog post lives in the `_posts` folder with a naming convention: `YYYY-MM-DD-title.md`

### Front Matter

Each post starts with YAML front matter:

```yaml
---
title: Your Post Title
date: 2026-06-05
categories: [category1, category2]
tags: [tag1, tag2]
---
```

### Markdown Content

Write your content in Markdown:

```markdown
# Your Heading

Your paragraph text here.

## Subheading

- List item 1
- List item 2
- List item 3
```
